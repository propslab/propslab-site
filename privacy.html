02 / 09 取得する情報

取得する情報

お問い合わせフォームを通じて、以下の情報を取得いたします。

  • お名前(必須)
  • 屋号・事業名(任意)
  • メールアドレス(必須)
  • お電話番号(任意)
  • ご相談内容の区分(必須)
  • 詳細・ご質問内容(必須)
  • ご希望の連絡方法(必須)

加えて、サイトの利用に伴い、以下の技術的情報を自動的に取得する場合があります。

  • IPアドレス、ブラウザの種類・バージョン、参照元URL、訪問日時、閲覧ページ

これらの技術的情報は Cloudflare Web Analytics により取得され、Cookie は使用しません。個人を特定する形では収集いたしません。

03 / 09 利用目的

利用目的

取得した情報は、以下の目的でのみ利用いたします。

  • お問い合わせへのご返信、ご相談対応
  • お見積もり・ご提案書の作成、契約準備、業務連絡
  • ご契約後の業務遂行、関連連絡
  • サイト改善のための統計的分析(個人を識別しない形で実施)

これらの目的を超えて利用することはございません。営業目的のメールマガジン送信や、第三者への販売・提供は行いません。

05 / 09 保存期間

保存期間

お問い合わせいただいた個人情報は、原則としてご返信・業務連絡が完了してから 2年間 保管した後、削除いたします。

ご契約に至った場合は、契約期間中および契約終了後も、税務・会計その他の法令で求められる期間、または継続的なフォローアップに必要な期間、保管いたします。

06 / 09 安全管理措置

安全管理措置

個人情報の漏えい、滅失、毀損を防ぐため、合理的な安全管理措置を講じます。

  • 受信用メールアカウントへのアクセス制限・二要素認証の設定
  • サイト全体での暗号化通信(HTTPS)の利用
  • 外部サービス利用時のアクセスキー・認証情報の適切な管理
  • 個人情報を業務目的を超えて閲覧・利用しないこと

08 / 09 Cookie について

Cookie について

当サイトでは、フォーム送信時のスパム防止のため、hCaptcha が Cookie を使用する場合があります。

アクセス解析に利用している Cloudflare Web Analytics は、Cookie を使用しません。

ブラウザの設定により Cookie を無効にすることもできますが、その場合、フォーム送信が正常に動作しないことがあります。

09 / 09 改定

改定

本ポリシーは、法令の改正やサービス内容の変更に応じて改定することがあります。重要な変更を行う場合は、本ページにて告知いたします。

改定後のポリシーは、本ページに掲載した時点から効力を生じます。
